# Quickstart

1. Clone the Primodium Developer repository:

```bash copy
git clone https://github.com/primodiumxyz/developer.git
```

2. Change your active directory:

```bash copy
cd developer/examples/WriteDemo/packages/contracts
```

3. Install the necessary packages:

```bash copy
pnpm i
```

4. Update namespaces in scripts (choose your own to avoid collision).

5. Build the project:

```bash copy
pnpm build
```

6. Update addresses and private keys in `.env`

```bash copy
cp .env.example .env
```

7. Test the project:

```bash copy
forge test
```

8. Do a dry-run of deployment:

```bash copy
forge script script/RegisterWriteDemoSystem.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http
```

9. Deploy the System:

```bash copy
forge script script/RegisterWriteDemoSystem.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http --broadcast
```

10. Use a script to delegate permission and interact with the system:

```bash copy
forge script script/BuildIronMine.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http --broadcast
```

## File Structure

The
[WriteDemo](https://github.com/primodiumxyz/developer/tree/main/examples/WriteDemo)
consists of four main files:

- System Contract:
  [`src/systems/WriteDemoSystem.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/WriteDemo/packages/contracts/src/systems/WriteDemoSystem.sol)
- System Tests:
  [`test/WriteDemoSystem.t.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/WriteDemo/packages/contracts/test/WriteDemoSystem.t.sol)
- Deployment Script:
  [`script/RegisterWriteDemoSystem.s.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/WriteDemo/packages/contracts/script/RegisterWriteDemoSystem.s.sol)
- Interaction Script:
  [`script/BuildIronMine.s.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/WriteDemo/packages/contracts/script/BuildIronMine.s.sol)
