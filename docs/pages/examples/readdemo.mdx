# Read Demo

The Primodium **ReadDemo** introduces the basics of reading data from Primodium tables using the [MUD World Extension](https://mud.dev/guides/extending-a-world) system.

To setup a new world extension from template, follow the instructions in the [setup guide](../world-extension/setup.mdx). The source code for the ReadDemo is available on Github below:

- https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo
