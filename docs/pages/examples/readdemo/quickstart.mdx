# Quickstart

1. Clone the Primodium Developer repository:

```bash copy
git clone https://github.com/primodiumxyz/developer.git
```

2. Change your active directory:

```bash copy
cd developer/examples/ReadDemo/packages/contracts
```

3. Install the necessary packages:

```bash copy
pnpm i
```

4. Update namespaces in scripts (choose your own to avoid collision).

5. Build the project:

```bash copy
pnpm build
```

6. Update addresses and private keys in `.env`

```bash copy
cp .env.example .env
```

7. Test the project:

```bash copy
forge test
```

8. Do a dry-run of deployment:

```bash copy
forge script script/RegisterReadDemoSystem.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http
```

8. Deploy the System:

```bash copy
forge script script/RegisterReadDemoSystem.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http --broadcast
```

9. Use a script to interact with the system:

```bash copy
forge script script/ReadMainBaseLevel.s.sol --fork-url https://primodium-sepolia.rpc.caldera.xyz/http --broadcast
```

## File Structure

The
[ReadDemo](https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo)
consists of four main files:

- System Contract:
  [`src/systems/ReadDemoSystem.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo/packages/contracts/src/systems/ReadDemoSystem.sol)
- System Tests:
  [`test/ReadDemoSystem.t.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo/packages/contracts/test/ReadDemoSystem.t.sol)
- Deployment Script:
  [`script/RegisterReadDemoSystem.s.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo/packages/contracts/script/RegisterReadDemoSystem.s.sol)
- Interaction Script:
  [`script/ReadMainBaseLevel.s.sol`](https://github.com/primodiumxyz/developer/tree/main/examples/ReadDemo/packages/contracts/script/ReadMainBaseLevel.s.sol)
